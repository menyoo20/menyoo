## 🔍 Troubleshooting & Common Issues

### ❌ Menu Doesn't Open After Injection

**Cause:** Wrong toggle key or injection failure.
**Fix:**
1. Confirm ScriptHookV is installed and matches your game version.
2. Try pressing `F8` with Num Lock OFF.
3. Check `menyooConfig.xml` for the `MenuToggleKey` value and ensure it matches your keyboard layout.

---

### ❌ Game Crashes at Startup

**Cause:** Outdated ScriptHookV or .NET dependency mismatch.
**Fix:**
1. Update ScriptHookV to the latest release.
2. Reinstall .NET Framework 4.8 from Microsoft's site.
3. Delete `menyooConfig.xml` and let it regenerate fresh.

---

### ❌ Object Spooner Won't Save Scenes

**Cause:** Missing write permissions for the `menyooStuff` folder.
**Fix:**

```bash
# Windows — grant folder write access
icacls "%GAME_ROOT%\menyooStuff" /grant %USERNAME%:F /T

# macOS — fix permissions
chmod -R 755 ~/Library/Application\ Support/Menyoo/
```

---

### ❌ Features Greyed Out or Not Working

**Cause:** Running in an unsupported mode or missing dependencies.
**Fix:**
1. Ensure you are in Story Mode, not any online session.
2. Verify all required `.dll` files are in the root directory.
3. Restart the game with all overlays disabled.

---

### ❌ Android Overlay Not Appearing

**Cause:** Overlay permission not granted.
**Fix:**
1. Go to `Settings > Apps > [Companion App] > Display over other apps` → Enable.
2. Restart the companion app and re-inject.
